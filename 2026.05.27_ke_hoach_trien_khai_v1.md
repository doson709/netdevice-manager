# NetDevice Manager — Kế hoạch triển khai chi tiết

> **Ngày tạo:** 2026-05-27
> **Người tạo:** Mực (trợ lý)
> **Chủ đầu tư:** Đỗ Sơn

---

## 1. Tổng quan dự án

**Mục tiêu:** Xây dựng hệ thống quản lý thiết bị máy tính trong mạng nội bộ doanh nghiệp. Các máy client cài agent thu thập thông tin và gửi về server tập trung. Server hiển thị dashboard web để quản lý, theo dõi toàn bộ thiết bị.

**Quy mô:** Vài chục máy Windows, mạng nội bộ thông nhau, server nội bộ.

## 3. Stack công nghệ

| Component | Công nghệ | Lý do |
|---|---|---|
| **Agent (Client)** | Python 3.8+ | Chạy được trên Windows, dễ đóng gói .exe |
| **Agent → Packaging** | PyInstaller | Đóng gói thành file .exe, không cần cài Python trên client |
| **Backend API** | Python FastAPI | Nhẹ, nhanh, tự sinh Swagger docs |
| **Database** | SQLite (WAL mode) | Nhẹ, không cần cài đặt, chế độ WAL giúp ghi đồng thời an toàn |
| **Frontend** | React 18 + Tailwind CSS + Recharts | Dashboard hiện đại, responsive |
| **Server** | Uvicorn (ASGI) | Server FastAPI chính thức |
| **Realtime** | Polling (30s) | Đơn giản, đủ cho quy mô này |
| **Security** | API-Key Token Auth | Xác thực HTTP headers đảm bảo an toàn truyền dữ liệu nội bộ |

## 5. Chi tiết từng phần

### 5.1 Agent (Python — chạy trên máy client)

**Chức năng:** Thu thập thông tin thiết bị và gửi về server.

**Thông tin thu thập:**

| Nhóm | Thông tin chi tiết |
|---|---|
| **Hệ thống** | Tên máy, OS, phiên bản, kiến trúc, người dùng hiện tại |
| **CPU** | Model, số nhân vật lý/logical, tần số, % sử dụng |
| **RAM** | Tổng, đã dùng, còn trống, % sử dụng, swap |
| **Ổ đĩa** | Tên thiết bị, dung lượng, đã dùng, còn trống, % sử dụng |
| **Mạng** | IP, MAC (sử dụng Adapter vật lý hoạt động chính), subnet mask |
| **GPU** | Tên, dung lượng VRAM, driver version (đọc qua PowerShell/Registry) |
| **Mainboard** | Hãng, model, serial (đọc qua PowerShell để tương thích Win 11+) |
| **BIOS** | Hãng, phiên bản, ngày (đọc qua PowerShell để tương thích Win 11+) |
| **Phần mềm** | Tên, phiên bản, hãng sản xuất (từ Windows Registry) |
| **Tiến trình** | Top 20 tiến trình theo RAM (tên, PID, %RAM, %CPU) |
| **Windows** | Trạng thái kích hoạt, thời gian uptime |
| **Metadata** | Vị trí, phòng ban, người phụ trách, Device UUID sinh tự động |

**Thư viện sử dụng:**
- `psutil` — CPU, RAM, Disk, Network, Processes
- `requests` — Gửi HTTP POST đến server (kèm Header API Key xác thực)
- `winreg` — Đọc danh sách phần mềm từ Windows Registry
- `subprocess` + `PowerShell` / `winreg` — GPU, Mainboard, BIOS (thay thế cho `wmic` đã bị khai tử)
- `uuid` — Sinh UUID định danh duy nhất cho máy trạm (lưu lâu dài tại Registry hoặc config.json để tránh đổi ID khi thay card mạng)

**Cấu hình (config.json):**
```json
{
    "server_url": "http://192.168.1.100:8080",
    "secret_token": "secure-intranet-token-123",
    "device_uuid": "tự động sinh và lưu giữ trong lần đầu chạy",
    "report_interval": 60,
    "location": "Phòng IT, Tầng 2",
    "department": "IT",
    "owner": "Nguyễn Văn A"
}
```

**Tần suất gửi:** Mỗi 60 giây (có thể tùy chỉnh).

**Cách cài đặt:**
1. Cài Python 3.8+ trên máy client
2. Chạy `install.bat` → nhập IP server, vị trí, phòng ban, người dùng, token bảo mật
3. Agent tự động sinh UUID, lưu vào Registry/config.json, tự động chạy nền và khởi động cùng Windows

**Đóng gói thêm (tùy chọn):**
- Dùng PyInstaller đóng gói thành `netdevice-agent.exe` — không cần cài Python
- Whitelist thư mục cài đặt (`C:\Program Files\NetDeviceAgent`) trong antivirus policy hoặc ký tự chứng chỉ nội bộ (self-signed)
- Có thể dùng Task Scheduler thay vì Startup folder để chạy ẩn dưới quyền SYSTEM ổn định hơn

### 5.2 Backend API (FastAPI + SQLite)

**Cổng:** 8080 (có thể tùy chỉnh)

**Database schema (SQLite):**

```sql
-- Bảng thiết bị
CREATE TABLE devices (
    device_id       TEXT PRIMARY KEY,    -- UUID định danh máy khách duy nhất
    mac_address     TEXT,                -- Địa chỉ MAC vật lý chính
    hostname        TEXT,
    os_name         TEXT,
    os_version      TEXT,
    architecture    TEXT,
    current_user    TEXT,
    location        TEXT,
    department      TEXT,
    owner           TEXT,
    first_seen      DATETIME,
    last_seen       DATETIME,
    is_online       BOOLEAN DEFAULT 1
);

-- Bảng cấu hình phần cứng (lịch sử)
CREATE TABLE hardware_snapshots (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    device_id       TEXT REFERENCES devices(device_id),
    timestamp       DATETIME,
    cpu_model       TEXT,
    cpu_cores       INTEGER,
    cpu_usage       REAL,
    ram_total_gb    REAL,
    ram_used_gb     REAL,
    ram_usage       REAL,
    gpu_info        TEXT,               -- JSON
    motherboard     TEXT,               -- JSON
    bios            TEXT,               -- JSON
    uptime_seconds  REAL
);

-- Bảng ổ đĩa (lịch sử)
CREATE TABLE disk_snapshots (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    snapshot_id     INTEGER REFERENCES hardware_snapshots(id),
    device          TEXT,
    mountpoint      TEXT,
    total_gb        REAL,
    used_gb         REAL,
    free_gb         REAL,
    usage_percent   REAL
);

-- Bảng phần mềm
CREATE TABLE software (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    device_id       TEXT REFERENCES devices(device_id),
    name            TEXT,
    version         TEXT,
    publisher       TEXT,
    discovered_at   DATETIME,
    UNIQUE(device_id, name)
);

-- Bảng mạng
CREATE TABLE network_snapshots (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    snapshot_id     INTEGER REFERENCES hardware_snapshots(id),
    device_id       TEXT,
    adapter_name    TEXT,
    ip_address      TEXT,
    mac_address     TEXT,
    netmask         TEXT
);
```

**API Endpoints:**

| Method | Endpoint | Mô tả |
|---|---|---|
| `POST` | `/api/report` | Agent gửi dữ liệu định kỳ |
| `GET` | `/api/dashboard` | Tổng quan: tổng máy, online/offline, cảnh báo |
| `GET` | `/api/devices` | Danh sách thiết bị (filter, search, sort) |
| `GET` | `/api/devices/{id}` | Chi tiết 1 thiết bị |
| `GET` | `/api/devices/{id}/history` | Lịch sử tài nguyên (CPU, RAM theo thời gian) |
| `GET` | `/api/devices/{id}/software` | Danh sách phần mềm của 1 máy |
| `DELETE` | `/api/devices/{id}` | Xóa thiết bị |
| `PUT` | `/api/devices/{id}` | Cập nhật metadata (vị trí, phòng ban, owner) |
| `GET` | `/api/software/search` | Tìm phần mềm trên tất cả máy |
| `GET` | `/api/stats/os` | Thống kê theo OS |
| `GET` | `/api/stats/department` | Thống kê theo phòng ban |

**Logic quan trọng:**
- **Xác thực API Key**: Mọi HTTP POST từ Agent lên `/api/report` bắt buộc phải mang Header `X-API-Key` khớp với cấu hình phía server để chặn dữ liệu giả mạo từ mạng LAN.
- **Tối ưu ghi đồng thời (Concurrency)**: Bật chế độ WAL (Write-Ahead Logging) trên SQLite và set `timeout=30.0` để chống lỗi `database is locked` khi hàng chục Agent báo cáo cùng lúc.
- **Xử lý trùng lặp phần mềm (UPSERT)**: Khi lưu danh sách phần mềm cài đặt, sử dụng cú pháp `INSERT OR REPLACE` hoặc `INSERT ... ON CONFLICT(device_id, name) DO UPDATE SET version = EXCLUDED.version` để cập nhật phiên bản thay vì crash Unique constraint.
- **Quy trình báo cáo**: Agent gửi lên → server kiểm tra `device_id` (UUID) đã tồn tại chưa:
  - Chưa → INSERT vào `devices` + các bảng liên quan.
  - Rồi → UPDATE `last_seen`, INSERT snapshot mới.
- Nếu `last_seen` > 5 phút → đánh dấu `is_online = false`.
- Cleanup snapshot cũ (giữ 30 ngày gần nhất để tiết kiệm dung lượng).

## 7. Deployment

### Server (máy chủ nội bộ)

```bash
# 1. Cài Python 3.8+
# 2. Clone project
cd netdevice-manager/server

# 3. Tạo virtual environment
python -m venv venv
venv\Scripts\activate

# 4. Cài dependencies
pip install -r requirements.txt

# 5. Build frontend (nếu cần)
cd ../frontend
npm install
npm run build
# Copy dist/ vào server/static/

# 6. Chạy server
cd ../server
python main.py
# Server chạy tại http://<IP-máy-chủ>:8080
```

### Agent (mỗi máy client)


## 9. Rủi ro & mitigation

| Rủi ro | Mức độ | Giải pháp |
|---|---|---|
| Firewall chặn kết nối agent → server | Trung bình | Document rõ cách mở port, test kết nối trước khi deploy |
| Agent bị antivirus chặn (do đóng gói exe) | Trung bình | Whitelist thư mục cài đặt (`C:\Program Files\NetDeviceAgent`) hoặc ký số tự sinh nội bộ (Self-signed cert). |
| SQLite bị khóa khi nhiều agent ghi đồng thời | Trung bình | Bật WAL mode, thiết lập connection busy_timeout = 30.0s trong FastAPI SQLAlchemy/sqlite3. |
| Python chưa cài trên máy client | Trung bình | Cung cấp .exe đã đóng gói bằng PyInstaller (đã xử lý whitelist AV). |
| Thông tin phần mềm quá nhiều (hàng trăm phần mềm) | Thấp | Tích hợp Phân trang (Pagination) ở API backend và render ảo hóa (Virtual List) ở frontend để tối ưu hiệu năng. |
| WMIC bị Microsoft khai tử trên Windows 11+ | Trung bình | Thay thế việc gọi `wmic` bằng PowerShell hoặc đọc khóa Windows Registry trực tiếp. |
| Giả mạo hoặc đánh cắp dữ liệu trong mạng LAN | Trung bình | Bắt buộc gửi Token xác thực `X-API-Key` trong header mỗi request, khuyến nghị bật HTTPS nội bộ cho FastAPI. |
